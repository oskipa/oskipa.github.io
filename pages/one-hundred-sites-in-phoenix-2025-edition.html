    App 5: JavaScript Loto Numbers

    What I learned

    It is easier to add jquery into the app.

    1. Download the library and save it under assets/vendor/jquery.js
    2. Add it under assets/js/app.js via typscript

    import $ from '../vendor/jquery' window.$ = $ window.JQuery = $;

    Then in the main template I had to take out the defer on the script tag. Otherwise it wouldn't know that the library was there.

    And that was it.
